## 设计要点

| 元素 | 说明 |
|------|------|
| 色彩 | 主色 #2563eb（蓝色），背景 #f8fafc，不使用大红大绿促销色 |
| 字体层级 | 标题 24px 粗体 / 正文 15px 常规 / 辅助 13px 灰 |
| 动效 | 勾选标记用 CSS 缩放动画（0.5x -> 1x, 0.3s ease-out） |
| 签名 | 手写体或等宽字体，附日期，给人以"这是一封信"的感觉 |
| 分享区 | 放在最底部，默认收起，用户可按需展开——不要在庆祝屏上直接放分享按钮 |

## 关键措辞原则

1. **不用 "恭喜你购买了 Pro"** —— "购买"太商业，"你已经是 Pro 了"像欢迎加入俱乐部
2. **不用 "终身五折限时优惠"** —— 感谢页不谈钱，钱的部分在支付前已经说清楚了
3. **不用 "好评返现"** —— 破坏信任感
4. **多用 "你"** —— 通篇站在用户视角：你已经拥有了什么、接下来可以做什么
5. **适当示弱** —— "一个人做的"、"还不够完美" 反而增加可信度

## 压力测试

在发布前，用以下 3 种心态阅读感谢页文案：

- 我是一个刚付费但有点犹豫的用户 —— 文案是否能让我觉得"这笔钱花对了"？
- 我是一个不懂技术的普通用户 —— "价格走势图"这种词我能理解吗？
- 我是一个来挑刺的用户 —— 文案有没有任何虚假承诺或夸大其词？

调整完毕后，正式上线。
